## Getting Started

### Prerequisites
- Node.js 18+ 
- pnpm (recommended) or npm

### Installation

1. **Clone the repository**
   ```bash
   git clone <your-repo-url>
   cd frontend
   ```

2. **Install dependencies**
   ```bash
   pnpm install
   # or
   npm install
   ```

3. **Run the development server**
   ```bash
   pnpm dev
   # or
   npm run dev
   ```

4. **Open your browser**
   Navigate to [http://localhost:3000](http://localhost:3000)

### Build for Production

```bash
pnpm build
pnpm start
```

## Customization

### Personal Information

Update the following files with your information:

- **`app/layout.tsx`** - Site metadata, title, description
- **`components/Hero.tsx`** - Name, description, social links
- **`components/Header.tsx`** - Navigation links
- **`components/Footer.tsx`** - Contact information

### Content Updates

- **Articles**: Edit `app/articles/page.tsx` and `app/articles/[slug]/page.tsx`
- **Projects**: Update `app/projects/page.tsx`
- **Speaking**: Modify `app/speaking/page.tsx`
- **Work Experience**: Edit `components/WorkTimeline.tsx`
- **Uses**: Update `app/uses/page.tsx`

### Images

Replace placeholder images in the `public/` directory:

- `avatar.jpg` - Your profile picture (128x128px or larger)
- `photo1.jpg` through `photo6.jpg` - Photo rail images (square aspect ratio)
- `project1.jpg` through `project4.jpg` - Project screenshots (16:9 aspect ratio)

### Styling

- **Colors**: Modify CSS custom properties in `app/globals.css`
- **Layout**: Adjust Tailwind classes in component files
- **Typography**: Update font variables in `app/layout.tsx`

### Social Links

Update social media links in:
- `components/Hero.tsx` - Hero section social icons
- `components/Footer.tsx` - Footer social links
